# 温控调校

**温控调校**页面公开了一些通常不会向用户展示的底层温控管理开关。

## 分区说明

- **通用**
  控制应用是否接收热状态更新，以及系统在发热时是否主动降低功耗。
- **口袋高温模式（HIP）**
  在支持的 iPhone 上，控制屏幕熄灭且设备放入口袋时的温控策略。
- **阳光暴晒**
  在自动检测与手动模式之间切换，并展示当前的阳光暴露状态。
- **温控等级**
  展示并允许手动调整当前的温控压力与通知等级，通过彩色分段条进行控制。

## 开关说明

- **🔒锁定图标**：阻止 `thermalmonitord` 将该设置恢复为默认值。当你需要让自定义值在重启或系统调整后继续生效时开启。
- **✅勾选开关**：功能本身的开/关状态。根据需要而启用或关闭具体行为。

## 安全提示

修改这些设置可能会：

- 提升机身与电池温度；
- 影响充电、背光、手电筒与多媒体行为。

如非充分了解风险，请谨慎使用。

## 说明

- 通过 Battman 修改**温控调校**配置会直接更改系统的全局温控设置。iOS 会将这些设置视为用户数据，备份时会被保存；如果将备份恢复到另一台设备，同样的温控设置也会随之应用。
