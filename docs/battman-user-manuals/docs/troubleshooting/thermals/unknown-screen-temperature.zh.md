# 「未知」屏幕温度

有时 Battman 无法将系统的读数映射到有意义的屏幕或面板温度。在这种情况下，界面可能会在屏幕温度处显示**“未知”**，或者完全省略该数值。

## 可能原因

- 设备固件并未提供单独的面板温度传感器。
- 当前 iOS 版本隐藏或聚合了相关传感器。
- Battman 运行在模拟器或不受支持的平台（例如某些 Mac）。
- Battman 尚未支持读取您设备的面板温度。

## 可以怎么处理

- 使用**硬件温度**页面查看其他可用传感器。
- 检查异常的热压或降频现象，而不是依赖单一的屏幕温度数值。
- 保持 Battman 为最新版本；可能会随着时间的推移添加对更多传感器的支持。
