# 标示位

Battman 显示由电池电量计芯片直接上报的原始标示位，这些位来自德州仪器的电量计（bq274xx、bq275xx、bq20zxx），在**标示位**行按高字节和低字节各一行呈现。

## 在 App 里你会看到

- 一个十六进制数（如 `0x00CC`）表示完整的 16 位标示字。
- 两行分段开关显示各位：上行为高字节，下行为低字节。
- 高亮/填充的段表示该位被置位，灰色表示未置位。

## Battman 如何映射标示位

- 若能识别具体电量计型号（设备名或主板标识），Battman 会加载对应的布局；否则会选择最接近的布局。
- 两行展示全部位，点亮的段即为置位。
- 含义遵循德州仪器的数据手册，核心字段见下表。

## 常见标示位含义

**bq2742x（含 bq27421/25/26/27）**

高字节（界面左→右）

| 位 | 含义 |
| --- | --- |
| OT | 过温 |
| UT | 低温 |
| EEFAIL | EEPROM 写入失败（仅 27425） |
| FC | 检测到满电 |
| CHG | 允许充电 |

低字节（左→右）

| 位 | 含义 |
| --- | --- |
| OCVTAKEN | 已采集开路电压 |
| DODC | 放电深度校正进行中（421/426/427） |
| ITPOR | 上电复位发生 |
| CFGUPMODE | 处于配置更新模式 |
| BAT_DET | 检测到电池 |
| SOC1 | 达到低电量阈值 |
| SOCF | 达到危急电量阈值 |
| DSG | 放电中 |

**bq2754x / 通用 bq275**

高字节（左→右）

| 位 | 含义 |
| --- | --- |
| OTC | 充电过温（27545） |
| OTD | 放电过温（27545） |
| BATHI | 电压过高 |
| BATLOW | 电压过低 |
| CHG_INH | 因温度禁止充电 |
| FC | 检测到满电 |
| CHG | 允许充电（27545） |

低字节 — 通用 bq275（左→右）

| 位 | 含义 |
| --- | --- |
| CHG_SUS | 充电挂起 |
| IMAX | Imax 变化中断 |
| CHG | 允许充电 |
| SOC1 | 达到低电量阈值 |
| SOCF | 达到危急电量阈值 |
| DSG | 放电中 |

低字节 — bq27545 变体（左→右）

| 位 | 含义 |
| --- | --- |
| OCVTAKEN | 休眠开路电压采样 |
| ISD | 检测到内部短路 |
| TDD | 检测到极耳断开 |
| HW1 / HW0 | 器件 ID 位 |
| SOC1 | 达到低电量阈值 |
| SOCF | 达到危急电量阈值 |
| DSG | 放电中 |

**bq20zxx（MacBook 电池状态）**

高字节（左→右）

| 位 | 含义 |
| --- | --- |
| OCA | 过充告警 |
| TCA | 终止充电告警 |
| OTA | 过温告警 |
| TDA | 终止放电告警 |
| RCA | 剩余容量告警 |
| RTA | 剩余时间告警 |

低字节（左→右）

| 位 | 含义 |
| --- | --- |
| INIT | 电量计初始化 |
| DSG | 放电中 |
| FC | 满电 |
| FD | 完全放空 |
| EC3 | SBS 错误码位 3 |
| EC2 | SBS 错误码位 2 |
| EC1 | SBS 错误码位 1 |
| EC0 | SBS 错误码位 0 |

## 说明

- 标示位布局遵循 TI 文档；阈值和时序请参考对应数据手册。
- Battman 仅展示这些位；解读依赖充电状态与整机配置。如标示位长期与实际不符，可能是电池包或电量计上报异常。
