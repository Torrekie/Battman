# 警告條件

本頁說明 Battman 基於電池與系統報告的數據而發出警告的條件。

## **電池信息**中的數據警告

Battman 會標出可疑或丟失的**電池信息**監測數據，並在此列的側邊顯示一個「警告」圖標，輕按圖標可以顯示如下之一的詳細信息。

- **剩餘容量**
    - 當其報告數據超過了「滿充容量」，或者比設備計算的「真實剩餘容量」多出大於 10 mAh 時，報告異常數據信息：「電池剩餘容量數據異常，使用的可能是未經認證的電池部件。」，並且在其超過「滿充容量」時，警告信息中會顯示一個基於現有數據計算的「預估剩餘」。
    - 當數據丟失時，觸發「未獲取到剩餘容量。」
- **循環計數**
    - 當「循環計數」超出標稱設計目標時警告。如果電量計未提供設計值，Battman 對 iPhone 14 系列及更早機型使用 500，對 iPhone 15 系列及更新機型、iPad、Apple Watch 和 MacBook 使用 1000，對 iPod 使用 400。警告會提示電池容量或峰值效能可能降低。
- **耗盡時間**
    - Battman 會在裝置充電時隱藏「耗盡時間」，並拒絕不可用值或哨兵值。放電時，Battman 會將有效的報告值與使用浮點運算得出的容量/電流預估進行比較；差異過大時顯示「耗盡時間與目前的電池資料不一致。」
- **DOD₀ 參考值**
    - DOD₀ 是採用 0–16384 標度的無量綱原始參考值，不是容量，也不能與 Qmax 比較。高於文件規定範圍的值會顯示「DOD₀ 資料無效。」；範圍內的值不會觸發警告。

警告標題視特定條件可能為其中之一：「錯誤的數據」、「異常的數據」、「過大的數據」或者「空數據」。

如果你持續遇到這些警告，請考慮尋求專業硬件診斷。
