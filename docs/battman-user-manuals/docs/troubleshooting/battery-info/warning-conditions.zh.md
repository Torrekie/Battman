# 警告条件

本页说明 Battman 基于电池与系统报告的数据而发出警告的条件。

## **电池信息**中的数据警告

Battman 会标出可疑或丢失的**电池信息**监测数据，并在此列的侧边显示一个“警告”图标，轻按图标可以显示如下之一的详细信息。

- **剩余容量**
    - 当其报告数据超过了“满充容量”，或者比设备计算的“真实剩余容量”多出大于 10 mAh 时，报告异常数据信息：“电池剩余容量数据异常，使用的可能是未经认证的电池部件。”，并且在其超过“满充容量”时，警告信息中会显示一个基于现有数据计算的“预估剩余”。
    - 当数据丢失时，触发“未获取到剩余容量。”
- **循环计数**
    - 当“循环计数”超出标称设计目标时警告。如果电量计未提供设计值，Battman 对 iPhone 14 系列及更早机型使用 500，对 iPhone 15 系列及更新机型、iPad、Apple Watch 和 MacBook 使用 1000，对 iPod 使用 400。警告会提示电池容量或峰值性能可能降低。
- **耗尽时间**
    - Battman 会在设备充电时隐藏“耗尽时间”，并拒绝不可用值或哨兵值。放电时，Battman 会将有效的报告值与使用浮点运算得出的容量/电流预估进行比较；差异过大时显示“耗尽时间与当前电池数据不一致。”
- **DOD₀ 参考值**
    - DOD₀ 是采用 0–16384 标度的无量纲原始参考值，不是容量，也不能与 Qmax 比较。高于文档规定范围的值会显示“DOD₀ 数据无效。”；范围内的值不会触发警告。

警告标题视特定条件可能为其中之一：“错误的数据”、“异常的数据”、“过大的数据”或者“空数据”。

如果你持续遇到这些警告，请考虑寻求专业硬件诊断。
